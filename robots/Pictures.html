TKU I.C. Lab

The team is affiliated with Intelligent Control Laboratory (ICLab) of Tamkang University (TKU) in Taiwan and consists of 3 advisors and 7 postgraduate students. We are planning to use M-bot, which is combination of UR5 robot and mobile platform, to take participate in Amazon Picking Challenge 2016. Our main research interests include fuzzy system, intelligent control system, system on programmable chip (SOPC), visual servo, and computer vision.



              

TKU M-bot in ARC  

This dual arm robot is a new generation robot designed and fabricated by Intelligent Control Laboratory (ICLab) of Tamkang University (TKU) in Taiwan. The degree of freedom (DOF) in each arm are 7-DOF. In control system, this robot can detect and avoid the singularity automatically. In vision system, It can detect the target even in a very complex environment. This robot had joined the 3th Amazon Robotic Challenge and placed top 16 in the world.

M-bot  

M-bot is a mobile manipulator. It consists of a UR5 robot with 6 degree-of-freedom and an omnidirectional drive mobile platform. Robot Operating System (ROS) is adopted as the software framework so that we can develop whole system rapidly. To recognize and do pick-and-place task, we use SURF algorithm to detect object and use Moveit! to control UR5.

6-DOF Robot Manipulator (Bumble Bee-MIT)  

Bumble Bee-MIT is a high-quality programmable 6-DOF robot manipulator designed and fabricated by Intelligent Control Laboratory (ICLab) of Tamkang University (TKU) in Taiwan. Custom gripper can grab any kinds of items, and the gripper won't hurt the target. Bumble Bee-MIT can accomplish a lot of difficult tasks. Bumble Bee-MIT also awards great place at HIWIN Intelligent Robotics Implementation Competition several times in Taiwan.

  • Weight: 170 kg
  • Payload: 5 kg
  • Max. reach: 1000 mm
  • Number of Axes: 6
  • Repeat Accuracy: ±0.1 mm

7-DOF Robot Manipulator (Bumble Bee-EVO)  

Bumble Bee-EVO is a high-quality programmable 7-DOF robot manipulator designed and fabricated by Intelligent Control Laboratory (ICLab) of Tamkang University (TKU) in Taiwan. Custom gripper can grab any kinds of items, and the gripper won't hurt the target. Bumble Bee-EVO can accomplish a lot of difficult tasks.   • Repeat Accuracy: ±0.3 mm

FIRA RoboSot Soccer Robot  

Middle-size soccer robot is a wheeled mobile robot. The main purpose of this robot is to play soccer game. There are three features of this robot. First, the omnidirectional vision system. It can make robot get the information of whole soccer field. Second, dribbling device and kicking mechanism. Robot can dribble and kick the ball into the target door. Final, omnidirectional motion system. It let robot have ability to move any direction.

  • Birth: 2016
  • Size: 43*43*67 cm3
  • Weight: 16 kg
  • Drive: 3 WD
  • OS: Linux

TKU FEBOT  

Fire-fighting is a wheeled mobile robot. The main putpose of this robot is to extingush the fire. Fire-fighting robot can slef-patrol in the indoor environment. When robot discover the fire, it will move forware the fire and extingush the fire. There are three features of this robot. First, fire detecting system. Fire-fighting can discover the fire through thermal camera. Second, localization system. We use scan range finder to find robot's position through hector slam algorithm. Final, extinguishing system. When robot aim the fire, it will put out the fire by using fire extingusher.

  • Birth: 2016
  • Size: 43*43*83 cm3
  • Weight: 10 kg
  • Drive: 3 WD
  • OS: Linux
